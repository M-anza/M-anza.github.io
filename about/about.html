About

Throughout my web development journey, I have focused on building practical, user-friendly applications with clean design and seamless user experiences. I have developed several projects using modern frameworks and tools such as React, Next.js, Material-UI, Expo, and Expo Go. Some of my key projects include a Memory Game that tests concentration and recall, a Recipe Search App with dynamic ingredient filtering and real-time API interaction, a recently deployed Match the Animals Game showcasing my interactive UI and state management skills, and a Tilde mock-up demonstrating my ability to design sleek, responsive, and highly usable interfaces. These projects have strengthened my skills in component-based architecture, Redux state management, and front-end testing using tools like React Testing Library and TDD Testing Library. My experience with mobile development through Expo has also expanded my ability to build smooth, cross-platform applications. I enjoy tackling new challenges, improving code quality, and creating engaging, accessible, and efficient web solutions that provide real value to users.
