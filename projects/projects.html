🐾 Match the Animals click here

🎮 Game Instructions:

  • Flip cards one by one to reveal the animals.
  • Find matching pairs of the same animal.
  • Keep flipping until you’ve matched all the animals!

⚙️ Game Settings:

  • Choose your preferred board size before you begin or adjust it anytime.
  • Resize the board to make the game easier or more challenging.

🛠️ Frameworks & Technologies Used:

  • React.js – for building interactive UI components.
  • Material-UI – for responsive design and consistent styling.
  • CSS Modules – for scoped component styling.
  • React Testing Library – for writing unit tests using TDD.
  • Vercel – for seamless project deployment.

🎓 Purpose:

"Match the Animals" was built to demonstrate my understanding of React fundamentals, test-driven development (TDD), and responsive UI design.
